# Base-N Number Conversion

## Problem Statement
This problems does the `to_binary()` to `to_base_n()`. The choice was between entering a number between 2 and 16. If there was a number outside of that then it would show error. 

## Describe the Solution
The solution is it prints digits between 0-9 and A-F for hexadecimal as in 0x. prefixes are octal (0) and hexadecimal as (0x).

## Pros and Cons of the Solution

### Pros
-  2-16 works
- hexadecimal is done well
- The code runs


### Cons
- Recursion probably does not work with big numbers
- Does not support negative numbers
